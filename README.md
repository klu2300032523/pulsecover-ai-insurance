# PulseCover – AI-Powered Insurance for Gig Workers 🚀

## 📌 Overview
PulseCover is a parametric insurance platform designed for food delivery workers in India.

It protects workers from income loss caused by:
- Weather disruptions
- Traffic congestion
- Platform outages

## 💡 Unique Idea
PulseCover uses a multi-factor AI risk engine combining:
- Weather data
- Traffic data
- Platform downtime simulation
- Hyperlocal risk scoring

## 🧠 Features
- Weekly premium calculation
- AI-based risk estimation (mock)
- Parametric triggers (rain, traffic)
- Instant payout simulation
- Simple onboarding UI

## ⚙️ Tech Stack
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Open `frontend/index.html`
2. Enter income
3. Click "Get Premium"
4. Simulate disruptions

## 🔄 Workflow
1. User registers
2. Premium calculated weekly
3. External triggers monitored
4. Automatic payout released

- Real weather API integration
- ML-based premium prediction
