# 🚀 PulseCover – AI-Powered Insurance for Gig Workers

## 📌 Problem Statement
Gig workers (like Swiggy/Zomato delivery partners) often lose income due to external disruptions such as bad weather, traffic congestion, or platform outages.

This project builds an **AI-powered parametric insurance system** that automatically compensates workers for income loss.

---

## 💡 Our Solution
PulseCover provides a **smart income protection system** that:

- Calculates weekly insurance premium
- Monitors real-time external disruptions
- Automatically triggers payouts
- Ensures zero manual claim process

---

## 🎯 Unique Innovation
Unlike basic solutions, PulseCover uses a **multi-trigger AI model**:

- 🌧 Weather disruptions (rain, heat)
- 🚦 Traffic congestion
- 📱 Platform downtime simulation
- 📍 Hyperlocal risk scoring

👉 This creates a more realistic and intelligent insurance system.

---

## 🧠 Key Features

- ✅ Weekly premium calculation (aligned with gig worker income cycle)
- ✅ AI-based risk estimation (mock logic)
- ✅ Parametric triggers (rain, traffic)

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the `frontend` folder
3. Open `index.html` in browser OR use Live Server
4. Enter weekly income
5. Click "Get Premium"
6. Simulate disruption → view payout

---

## 🔄 Workflow

1. User enters income
2. System calculates weekly premium
3. External disruptions are monitored
4. If disruption occurs → automatic payout triggered
